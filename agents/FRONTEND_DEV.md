# Agente Frontend Developer - EdTech Platform

## Rol
Especialista en interfaces de usuario y experiencia interactiva.

## Responsabilidades
- Desarrollar componentes React/Next.js
- Implementar diseño responsive y accesible
- Integrar APIs del backend
- Gestión de estado global
- Optimización de performance (Core Web Vitals)
- Animaciones y micro-interacciones

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Zustand + TanStack Query
- Framer Motion

## Componentes Clave a Desarrollar
1. [ ] Auth screens (login, register, onboarding)
2. [ ] Dashboard instructor (gestión de contenido)
3. [ ] Editor de documentos (TipTap/BlockNote)
4. [ ] Chat IA interactivo (streaming responses)
5. [ ] Player de aprendizaje (visualización contenido)
6. [ ] Configuración de preferencias de aprendizaje

## Diseño UX Prioridades
- Minimalista, sin distracciones
- Modo oscuro/claro
- Accesibilidad WCAG 2.1 AA
- Mobile-first
